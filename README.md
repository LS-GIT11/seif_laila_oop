# Pokemon Starters OOP

An object-oriented programming assignment implementing Pokemon starters in PHP.

## Installation

1. Clone the repository
2. Place the folder in your web server's document root
3. Open the index.php file in your browser

## Usage

Open index.php in a web browser to view the Pokemon starters application.

## Rubric checklist

### (MMED-3014)

- Classes have three levels of hierarchy: 5 points
- Three methods describing object's properties or actions: 2 points
- Three properties describing objects: 1 point
- Trait to share functionality: 2 points
- Documentation and type-hints: 3 points
- index.php file with autoloader, loading classes: 2 points
